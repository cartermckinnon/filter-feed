filter-feed.me
What is this?

Sometimes a feed (such as a podcast) contains items that you aren't interested in. Or, you might want to sort items in one feed into several different feeds. You might just want to modify the feed's metadata.


This site lets you define a set of filters and overrides for a feed, and gives you a URL for the resulting feed, that you can use in your RSS reader or podcast player.
